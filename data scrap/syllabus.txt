Here is a breakdown of the subjects, chapters, and subtopics from the GATE 2026 Computer Science and Information Technology syllabus:

Section 1: Engineering Mathematics

Discrete Mathematics:

Logic: Propositional and first order logic.

Sets, Relations, and Functions: Sets, relations, functions, partial orders and lattices.

Algebraic Structures: Monoids, Groups.

Graph Theory: Connectivity, matching, colouring.

Combinatorics: Counting, recurrence relations, generating functions.

Linear Algebra:

Matrices and Determinants: Matrices, determinants.

System of Linear Equations: System of linear equations.

Eigenvalues and Eigenvectors: Eigenvalues and eigenvectors.

Decomposition: LU decomposition.

Calculus:

Limits and Continuity: Limits, continuity and differentiability.

Extrema: Maxima and minima.

Theorems: Mean value theorem.

Integration: Integration.

Probability and Statistics:

Random Variables and Distributions: Random variables, Uniform, normal, exponential, Poisson and binomial distributions.

Descriptive Statistics: Mean, median, mode and standard deviation.

Conditional Probability: Conditional probability and Bayes theorem.

Section 2: Digital Logic

Boolean Algebra

Circuits: Combinational and sequential circuits.

Minimization

Number Representations: Number representations and computer arithmetic (fixed and floating point).

Section 3: Computer Organization and Architecture

Instruction Set Architecture: Machine instructions and addressing modes.

Central Processing Unit (CPU): ALU, data-path and control unit.

Pipelining: Instruction pipelining, pipeline hazards.

Memory Hierarchy: Cache, main memory and secondary storage.

Input/Output (I/O): I/O interface (interrupt and DMA mode).

Section 4: Programming and Data Structures

Programming Language: Programming in C.

Recursion

Linear Data Structures: Arrays, stacks, queues, linked lists.

Non-Linear Data Structures: Trees, binary search trees, binary heaps, graphs.

Section 5: Algorithms

Basic Techniques: Searching, sorting, hashing.

Complexity Analysis: Asymptotic worst case time and space complexity.

Algorithm Design Techniques: Greedy, dynamic programming and divide-and-conquer.

Graph Algorithms: Graph traversals, minimum spanning trees, shortest paths.

Section 6: Theory of Computation

Formal Languages and Automata: Regular expressions and finite automata, Context-free grammars and push-down automata.

Language Properties: Regular and context-free languages, pumping lemma.

Computability: Turing machines and undecidability.

Section 7: Compiler Design

Phases of Compilation: Lexical analysis, parsing, syntax-directed translation.

Runtime Environments

Intermediate Code: Intermediate code generation.

Optimization: Local optimization, Data flow analyses: constant propagation, liveness analysis, common sub expression elimination.

Section 8: Operating System

Core Concepts: System calls, processes, threads.

Concurrency: Inter-process communication, concurrency and synchronization, Deadlock.

Scheduling: CPU and I/O scheduling.

Memory Management: Memory management and virtual memory.

File Systems

Section 9: Databases

Data Models: ER-model, Relational model: relational algebra, tuple calculus, SQL.

Database Design: Integrity constraints, normal forms.

File Organization and Indexing: File organization, indexing (e.g., B and B+ trees).

Transactions: Transactions and concurrency control.

Section 10: Computer Networks

Network Models: Concept of layering: OSI and TCP/IP Protocol Stacks.

Switching: Basics of packet, circuit and virtual circuit-switching.

Data Link Layer: Framing, error detection, Medium Access Control, Ethernet bridging.

Network Layer:

Routing: Routing protocols: shortest path, flooding, distance vector and link state routing.

IP Addressing: Fragmentation and IP addressing, IPv4, CIDR notation.

IP Support Protocols: Basics of IP support protocols (ARP, DHCP, ICMP), Network Address Translation (NAT).

Transport Layer: Flow control and congestion control, UDP, TCP, sockets.

Application Layer: Application layer protocols: DNS, SMTP, HTTP, FTP, Email.
